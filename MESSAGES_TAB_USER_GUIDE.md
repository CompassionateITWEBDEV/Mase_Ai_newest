## â“ **WHAT IF...**

### **Q: No messages showing?**
**A:** Empty state will show with "Compose Your First Message" button

### **Q: Filter shows (0)?**
**A:** No messages in that category
- Try "All" filter to see everything
- Or compose a new message

### **Q: Can't send message?**
**A:** Check if:
- Subject is filled in
- Message content is filled in
- Both are required!

### **Q: Message not sending?**
**A:** 
1. Check internet connection
2. Try again in a few seconds
3. Click refresh to see if it went through

### **Q: How to find old messages?**
**A:** 
- Click "All" filter
- Scroll down (most recent at top)
- All messages are preserved
