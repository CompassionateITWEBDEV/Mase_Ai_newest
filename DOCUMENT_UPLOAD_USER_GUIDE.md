## **ğŸ†˜ Troubleshooting**

### **"No Upload button visible"**
**Solution:** Make sure you're on the **"Referral Tracker"** tab

### **"File too large"**
**Solution:** Files must be under 10MB. Compress or split large files

### **"Invalid file type"**
**Solution:** Only upload PDF, DOC, DOCX, JPG, PNG, GIF, or TXT files

### **"Upload failed"**
**Possible causes:**
1. Network connection lost
2. File too large
3. Invalid file type
4. Server error

**Try:**
1. Check internet connection
2. Verify file size and type
3. Try again in a few seconds
4. Contact support if persists

### **"Documents not showing in modal"**
**Solution:** 
1. Close and reopen the details modal
2. Click refresh button in tracker
3. Wait a few seconds and try again

### **Multiple Files Upload**

### **Image Documents**
```
For images (insurance cards, photos):
- Take clear, well-lit photos
- Make sure text is readable
- JPEG/PNG formats work best
```

### **PDF Documents**
```
For PDF files:
- Use PDF instead of Word when possible (smaller file size)
- Combine related documents into one PDF (easier to manage)
```

### **Organization**
```
Use document types consistently:
- Medical = Clinical documents
- Insurance = Payment/coverage documents  
- Consent = Legal/permission documents
- Other = Everything else
```
